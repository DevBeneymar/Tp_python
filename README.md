📘 TP Python

Ce dépôt contient une série de travaux pratiques (TP) réalisés en Python dans le but de renforcer les compétences en programmation et en résolution de problèmes.

🎯 Objectifs
1. Comprendre les bases du langage Python
2. Manipuler les structures de données (listes, dictionnaires, tuples)
3. Appliquer les concepts fondamentaux (conditions, boucles, fonctions)
4. Développer une logique algorithmique solide

🛠️ Technologies utilisées
Python 3.x
